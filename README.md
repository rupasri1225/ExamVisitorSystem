# ExamVisitorSystem
🏫 Exam Visitor System

🚀 Description

Exam Visitor System is a web-based application designed to manage and monitor visitors during examinations. It helps institutions maintain records of visitors, ensure security, and streamline the entry and exit process efficiently.

---

🎯 Features

- 📌 Visitor registration system
- 📝 Entry and exit tracking
- 🔍 Search and filter visitor records
- 🔐 Secure data handling
- 📊 Simple and user-friendly interface
